Ters dönüşüm yöntemiyle büyük F bulunamıyorsa ilk kullanacağımız yöntem kabul-red yöntemidir.
Sürekli rastgele değişkenler için kabul-red algoritması:
1. g dağılımına sahip rastgele bir Y değişkeni oluşturun.
2. U'yu (Y'den bağımsız olarak) üretin.
3. Eğer u<=f(y)/c*g(y) ise X=Y'yi ("kabul et"), aksi takdirde 1'e ("reddet") geri dön.
